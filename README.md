### 解决什么问题?
- 下载妹子图 http://www.mzitu.com/
- 下载妹子图，并且存入硬盘

### 如何开始？
- 手动方式
	- git clone https://github.com/owen-carter/spider-mzitu.git
	- cd spider-mzitu
	- npm install
	- npm run app
- 自动
```bash
curl -o- https://raw.githubusercontent.com/owen-carter/spider-mzitu/master/start.sh | bash
```
